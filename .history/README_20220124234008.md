###

## 窗口程序的基本框架

```python
    import wx
    app = wx.App()# 实例化wxpython对象
    frame = wx.Frame(None, -1, "Hello, World!")# 主窗口
    frame.Show(True)# 显示
    app.MainLoop()# 
```

>如果工作在windows平台的话，建议同时安装pywin32模块。pywin32允许你像VC一样的使用python开发win32应用，更重要的是，可以用它直接操控win32程序，捕捉当前窗口、获取焦点等
